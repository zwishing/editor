# Refine Dropdown Styles

| ID                        | Status    | Category        |
| :------------------------ | :-------- | :-------------- |
| **refine-dropdown-style** | **Draft** | **Enhancement** |

## Summary

Customize the style of `Select` and `Combobox` components to place the selected checkmark on the right, ensure consistent focus effects, and match hover background colors with the selected layer style.

## Problem Statement

The user prefers a specific visual style for dropdowns that differs from the default left-aligned checkmark, and wants specific hover colors.

## Goals

1.  Move checkmark to the right for `Select` and `Combobox` items.
2.  Ensure focus styles on dropdown triggers match standard inputs.
3.  Update dropdown item hover/selection background to match "selected layer" color (lighter).
