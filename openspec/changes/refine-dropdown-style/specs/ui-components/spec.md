# Dropdown Style Spec

## Requirements

### Requirement: Checkmark Position

The checkmark indicating the selected item in `Select` and `Combobox` components MUST be positioned on the right side of the item.

### Requirement: Item Padding

The items MUST NOT have left padding reserved for the checkmark if the checkmark is moved to the right.

### Requirement: Focus Style

The dropdown trigger MUST have a focus style consistent with other input fields (e.g., ring/border).

### Requirement: Hover Background

The background color of a hovered or actively selected item in the dropdown list MUST be lighter, matching the "selected layer" background color (e.g., `bg-accent` or similar lighter variant).
