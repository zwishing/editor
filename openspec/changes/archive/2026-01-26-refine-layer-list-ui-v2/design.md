# Design

## Dimensions

- **Layer List Width**: Increase width (e.g., from `240px`/`default` to `300px` or `320px`).
- **Header Height**: Standardize to `h-10` or `h-12` for both Layer List and Layer Editor.
- **Divider**: `border-b` with a light color (e.g., `border-panel-border` or `border-neutral-200`) separating header from content.

## Icons

- **Visibility**:
  - `visible`: Show icon only on hover (current behavior).
  - `none` (hidden): Show "eye-off" icon ALWAYS (persistent).

## Typography

- **Headings**: `text-sm font-semibold` or `text-base font-bold`, consistent across both panels.
