# Theme Style Spec

## Requirements

### Requirement: Primary Color

The application's primary color MUST be updated to `rgb(84, 179, 81)` (approx HSL `118.2 37.7% 51%`).

### Requirement: Consistency

This color MUST be applied to:

- `--primary`
- `--ring`
- Relevant sidebar accent/ring variables if they follow the brand color.
