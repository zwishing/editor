# Refine Cursor Style

| ID                      | Status    | Category        |
| :---------------------- | :-------- | :-------------- |
| **refine-cursor-style** | **Draft** | **Enhancement** |

## Summary

Ensure that clickable elements (dropdown triggers and items) display a pointer cursor on hover.

## Problem Statement

Currently, some interactive elements like dropdown triggers or items may display the default arrow cursor, which can make them feel less interactive or clickable.

## Goals

1.  Update `Select` and `Combobox` components to use `cursor-pointer` for triggers and items.
